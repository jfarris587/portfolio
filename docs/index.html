Beloved Transportation

Android Application

Beloved Transportation is a non-profit that helps the disabled with transportation to their medical appointments. As the Lead Developer, two other team members and I designed and developed an Android app that help drivers have readily available information about their clients, as well as GPS navigation to their appointments. In addition, we developed a web portal for the office to schedule their clients for their transportation services.


Weather+

React App

Weather+ is a React JS web application that gives users full detailed information about weather in their area. The app implements geolocation so the user can automatically have information about their current location. Openweather API powers the data, and users can search for any city in the United States for weather data and forecasts.


Places.

React App

Places. is a React JS app in a beta version. With an integration of a RESTAPI, it gives information on countries around the globe. Future versions with a more comprehensive API would update photos of each actual location, give list of tourist spots, and have potentially more interaction for the user.
